# Spotify Listening Behavior Analysis Dashboard | Power BI

## Overview

This project is an end-to-end Power BI dashboard developed to analyze Spotify listening history and uncover meaningful insights from user streaming behavior. The dashboard transforms raw listening data into interactive visualizations that highlight trends across albums, artists, tracks, and listening patterns.

The project demonstrates the complete Power BI workflow, including data cleaning, data modeling, DAX calculations, KPI development, and dashboard design.

---

## Business Objectives

The dashboard is designed to answer the following business questions:

### Album Analysis
- Track total albums played over time.
- Analyze yearly album listening trends.
- Compare weekday and weekend listening behavior.
- Identify the Top 5 most played albums.
- Perform Latest Year vs Previous Year comparison.
- Calculate Year-over-Year (YoY) growth.

### Artist Analysis
- Track artist listening trends over time.
- Analyze yearly artist diversity.
- Compare weekday and weekend artist listening.
- Identify the Top 5 most played artists.
- Compare Latest Year with Previous Year.
- Measure YoY growth.

### Track Analysis
- Monitor track listening trends.
- Analyze yearly track consumption.
- Compare weekday and weekend listening.
- Identify the Top 5 most played tracks.
- Compare Latest Year with Previous Year.
- Measure YoY growth.

### Listening Pattern Analysis
- Analyze listening activity by hour and day using a Heat Map.
- Identify peak listening periods.
- Compare Average Listening Time against Track Frequency using Quadrant Analysis.
- Classify tracks into engagement categories based on listening duration and frequency.

### Detailed Data Exploration
- Interactive grid containing album, artist, and track information.
- Drill-down and drill-up capabilities.
- Drill-through pages for detailed analysis.
- Export filtered data to CSV.

## Dataset

The dataset contains Spotify listening history and includes fields such as:

- Track Name
- Artist Name
- Album Name
- Timestamp
- Platform
- Playback Duration (Milliseconds Played)
- Reason Start
- Reason End
- Shuffle Status
- Skipped Status
- Spotify Track URI

These attributes enable detailed analysis of user listening behavior and engagement patterns.
